# Characters

Drop a portrait image (PNG, ideally with transparency) per character here. It's shown as a static
overlay while that character is speaking — no lip-sync, just a swap based on whose turn it is
(the same approach most "brainrot" videos use).

`speaker-a.png` / `speaker-b.png` are generated placeholder squares (solid color + letter), not
copyrighted character art. Swap in your own images — filename (minus extension) is what shows up
in the character picker, and should match the speaker names used in the script and in
`assets/voices/`.
